## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `boolean` | `false` | Two-way binding for switch state |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity of the switch |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of the switch |
| `disabled` | `boolean` | `false` | Disable the switch |
