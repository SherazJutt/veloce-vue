---
title: RangeSlider
description: A draggable range slider component for selecting numeric values
---

```javascript
import { RangeSlider } from "veloce/ui";
```

## Basic

::RangeSliderContent{:basic="true"}
::

::code-collapse
  ```vue
  <template>
    <RangeSlider v-model="value" />
  </template>
  
  <script setup lang="ts">
  import { RangeSlider } from "veloce-vue/ui";
  import { ref } from "vue";
  
  const value = ref(50);
  </script>
  ```
::

## Tooltip Position

::RangeSliderContent{:tooltipPosition="true"}
::

::code-collapse
```vue
<template>
  <div class="space-y-4">

      <p class="text-muted mb-2 text-sm">Tooltip Top</p>
```

::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `number` | `required` | Two-way binding for slider value (0-100) |
| `showTooltip` | `boolean` | `false` | Show tooltip with current value while dragging |
| `tooltipPosition` | `"top" \| "bottom"` | `"top"` | Position of the tooltip relative to the slider |
