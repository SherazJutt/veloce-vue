## Closable

::MessageContent{:closable="true"}
::

::code-collapse
```vue
<template>
  <div class="space-y-3">
    <Message v-if="showMessage1" severity="info" closable @close="showMessage1 = false"> 
      This message can be closed 
    </Message>
    <Message v-if="showMessage2" severity="success" closable @close="showMessage2 = false"> 
      Another closable message 
    </Message>
  </div>
</template>

<script setup lang="ts">
import { Message } from "veloce-vue/ui";
import { ref } from "vue";

const showMessage1 = ref(true);
const showMessage2 = ref(true);
</script>
```
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"info"` | Color severity of the message |
| `variant` | `"solid" \| "outlined"` | `"solid"` | Visual style variant |
| `icon` | `Component` | `null` | Custom icon component (defaults to severity icon) |
| `closable` | `boolean` | `false` | Show close button and emit close event |
| `margin` | `Margin` | `""` | Margin spacing |
| `marginTop` | `Margin` | `"lg"` | Top margin spacing |
| `marginBottom` | `Margin` | `"md"` | Bottom margin spacing |
| `padding` | `Padding` | `""` | Padding spacing |

## Events

| Event | Description |
|-------|-------------|
| `close` | Emitted when the close button is clicked |

## Slots

| Slot | Description |
|------|-------------|
| `default` | Message content |
| `icon` | Custom icon slot |
