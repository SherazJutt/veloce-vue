## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `""` | Button text label |
| `variant` | `"solid" \| "soft" \| "outlined" \| "ghost" \| "link"` | `"solid"` | Visual style variant |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Button size |
| `icon` | `Component` | `null` | Icon component to display |
| `iconPosition` | `"left" \| "right"` | `"right"` | Position of the icon relative to label |
| `rounded` | `boolean` | `false` | Apply full border radius (pill shape) |
| `loading` | `boolean` | `false` | Show loading state with spinner |
| `disabled` | `boolean` | `false` | Disable the button |
| `fontWeight` | `"normal" \| "medium" \| "semibold" \| "bold"` | `"medium"` | Font weight of the label |
| `neumorphic` | `boolean` | `false` | Apply neumorphic styling (requires soft variant) |
| `highlighted` | `boolean` | `false` | Apply highlight background |
