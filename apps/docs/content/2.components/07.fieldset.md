---
title: Fieldset
description: A container component for grouping form fields with optional legend and toggleable content
---

```javascript
import { Fieldset } from "veloce/ui";
```

## Basic

::FieldsetContent{:basic="true"}
::

::code-collapse
  ```vue
  <template>
    <Fieldset legend="Personal Information">
      <div class="space-y-4">
        <div>
          <label class="text-muted mb-1 block text-sm" for="name-field">Name</label>
          <Input id="name-field" v-model="name" placeholder="Enter your name" />
        </div>
        <div>
          <label class="text-muted mb-1 block text-sm" for="email-field">Email</label>
          <Input id="email-field" v-model="email" placeholder="Enter your email" />
        </div>
      </div>
    </Fieldset>
  </template>
  
  <script setup lang="ts">
  import { Fieldset, Input } from "veloce-vue/ui";
  import { ref } from "vue";
  
  const name = ref("");
  const email = ref("");
  </script>
  ```
::

## Toggleable

::FieldsetContent{:toggleable="true"}
::

::code-collapse
```vue
<template>
  <Fieldset v-model:collapsed="isCollapsed" legend="Advanced Options" :toggleable="true">
    <div class="space-y-4">
      <div>
        <label class="text-muted mb-1 block text-sm" for="option1-field">Option 1</label>
        <Input id="option1-field" v-model="option1" />
      </div>
      <div>
        <label class="text-muted mb-1 block text-sm" for="option2-field">Option 2</label>
        <Input id="option2-field" v-model="option2" />
      </div>
    </div>
  </Fieldset>
</template>

<script setup lang="ts">
import { Fieldset, Input } from "veloce-vue/ui";
import { ref } from "vue";

const isCollapsed = ref(false);
const option1 = ref("");
const option2 = ref("");
</script>
```
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model:collapsed` | `boolean` | `false` | Two-way binding for collapsed state (only works when `toggleable` is `true`) |
| `legend` | `string` | `""` | Legend text displayed in the fieldset header |
| `icon` | `Component` | `null` | Icon component to display before the legend |
| `toggleable` | `boolean` | `false` | Enable collapse/expand functionality |
