## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Current progress value |
| `max` | `number` | `100` | Maximum progress value |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity of the progress bar |
| `striped` | `boolean` | `false` | Apply striped pattern to progress bar |
| `animated` | `boolean` | `false` | Apply pulse animation to progress bar |
| `mode` | `"determinate" \| "indeterminate"` | `"determinate"` | Progress bar mode |
| `textColor` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral" \| "white"` | `"white"` | Color of the percentage text |
