## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `string` | `"100%"` | Width of the skeleton |
| `height` | `string` | `"1rem"` | Height of the skeleton |
| `variant` | `"text" \| "circular" \| "rectangular"` | `"text"` | Shape variant of the skeleton |
| `animation` | `"pulse" \| "wave" \| "none"` | `"pulse"` | Animation type |
| `rounded` | `boolean` | `false` | Apply rounded corners (automatically applied for circular variant) |
