## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `string` | `""` | Two-way binding for input value |
| `label` | `string` | `""` | Label text (used with `labelStyle="static"`) |
| `labelStyle` | `"float" \| "static"` | `"static"` | Label display style |
| `placeholder` | `string` | `""` | Placeholder text |
| `type` | `"text" \| "password"` | `"text"` | Input type |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of the input |
| `leadingIcon` | `Component` | `null` | Icon component to display on the left |
| `trailingIcon` | `Component` | `null` | Icon component to display on the right |
| `iconClass` | `string` | `""` | Custom CSS classes for icons |
| `helpText` | `string` | `""` | Help text displayed below the input |
