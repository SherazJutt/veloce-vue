```javascript
import { useToast } from "veloce/composables";
import { ToastContainer } from "veloce/toast";
```

## Setup

First, add the `ToastContainer` component to your app layout:

```vue
<template>
  <div>
    <!-- Your app content -->
    <ToastContainer position="bottom-right" :max-toasts="5" />
  </div>
</template>

<script setup lang="ts">
import { ToastContainer } from "veloce-vue/toast";
</script>
```

## Basic

::ToastContent{:basic="true"}
::

::code-collapse
  ```vue
  <template>
    <div class="flex flex-wrap gap-3">
      <Button label="Success Toast" severity="success" @click="showSuccess" />
      <Button label="Error Toast" severity="error" @click="showError" />
      <Button label="Warning Toast" severity="warning" @click="showWarning" />
      <Button label="Info Toast" severity="info" @click="showInfo" />
    </div>
  </template>
  
  <script setup lang="ts">
  import { Button } from "veloce-vue/ui";
  import { useToast } from "veloce-vue/toast";
  
  const toast = useToast();
  
  const showSuccess = () => {
    toast.success("Operation completed successfully!");
  };
  
  const showError = () => {
    toast.error("Something went wrong. Please try again.");
  };
  
  const showWarning = () => {
    toast.warning("Please check your input before submitting.");
  };
  
  const showInfo = () => {
    toast.info("New update available. Check it out!");
  };
  </script>
  ```

## Custom Duration

::ToastContent{:customDuration="true"}
::

::code-collapse
```vue
<template>
  <div class="flex flex-wrap gap-3">
    <Button label="Long Duration (10s)" severity="info" @click="showLongDuration" />
    <Button label="Short Duration (2s)" severity="warning" @click="showShortDuration" />
    <Button label="No Auto-Close" severity="error" @click="showNoAutoClose" />
  </div>
</template>

<script setup lang="ts">
import { Button } from "veloce-vue/ui";
import { useToast } from "veloce-vue/toast";

const toast = useToast();

const showLongDuration = () => {
  toast.info("This toast will stay for 10 seconds", { duration: 10000 });
};

const showShortDuration = () => {
  toast.warning("This toast disappears quickly", { duration: 2000 });
};

const showNoAutoClose = () => {
  toast.error("This toast won't auto-close. Click the X to dismiss.", { duration: 0 });
};
</script>
```

## Methods

The `useToast` composable provides the following methods:

| Method | Description |
|--------|-------------|
| `success(message, options?)` | Show a success toast |
| `error(message, options?)` | Show an error toast |
| `warning(message, options?)` | Show a warning toast |
| `info(message, options?)` | Show an info toast |
| `show(message, options?)` | Show a toast with custom options |
| `clear()` | Clear all toasts |

## ToastOptions

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"info"` | Color severity |
| `icon` | `Component` | `null` | Custom icon component |
| `duration` | `number` | `5000` | Duration in milliseconds (0 = no auto-close) |
| `closable` | `boolean` | `true` | Show close button |

## ToastContainer Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `"top-left" \| "top-right" \| "top-center" \| "bottom-left" \| "bottom-right" \| "bottom-center"` | `"top-right"` | Position of toast container |
| `maxToasts` | `number` | `5` | Maximum number of toasts to display simultaneously |
