## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `string` | `""` | Two-way binding for selected value |
| `options` | `string[]` | `required` | Array of option strings |
| `variant` | `"solid" \| "soft" \| "outlined" \| "ghost" \| "link"` | `"outlined"` | Visual style variant |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"neutral"` | Color severity |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of the select |
| `showFilter` | `boolean` | `false` | Show search filter input |
| `isOpen` | `boolean` | `false` | Control dropdown open state |
| `closeOnClickOutside` | `boolean` | `true` | Close dropdown when clicking outside |
| `fontWeight` | `"normal" \| "medium" \| "semibold" \| "bold"` | `"normal"` | Font weight of the label |

## Events

| Event | Description |
|-------|-------------|
| `update:isOpen` | Emitted when the dropdown open state changes |
