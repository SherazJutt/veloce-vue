# Accordion

The Accordion component is used to display collapsible content sections. Users can expand or collapse individual items to reveal or hide content. It supports single or multiple open items, custom icons, and flexible content rendering through slots.

## Installation

```javascript
import { Accordion } from "@veloce-vue/ui";
```

## Basic Usage

```vue
<template>
  <Accordion :items="items" />
</template>

<script setup lang="ts">
import { Accordion } from "@veloce-vue/ui";
import { type AccordionItem } from "@veloce-vue/types";

const items: AccordionItem[] = [
  {
    title: "Item 1",
    content: "This is the content for item 1. It can contain any text or HTML content.",
  },
  {
    title: "Item 2",
    content: "This is the content for item 2. You can add as many items as needed.",
  },
  {
    title: "Item 3",
    content: "This is the content for item 3. Each item can have different content.",
  },
];
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `AccordionItem[]` | **required** | Array of accordion items to display |
| `multiple` | `boolean` | `false` | Allow multiple items to be open simultaneously |
| `shadow` | `boolean` | `false` | Add shadow effect to the accordion container |
| `bordered` | `boolean` | `true` | Show border around accordion items |
| `rounded` | `"sm" \| "md" \| "lg" \| "xl" \| "none"` | `"sm"` | Border radius of the accordion container |
| `contentClass` | `string` | `""` | Custom CSS classes for accordion content |
| `headerClass` | `string` | `""` | Custom CSS classes for accordion headers |

## AccordionItem Type

Each item in the `items` array should follow the `AccordionItem` interface:

```typescript
interface AccordionItem {
  title: string;           // Required: The title text displayed in the header
  content: string;         // Required: The content text (ignored if slot is provided)
  icon?: Component;        // Optional: Vue component for the icon
  slot?: string;           // Optional: Slot name for custom content rendering
  active?: boolean;        // Optional: Initial active state of the item
}
```

### Custom Content with Slots

Use slots to render custom content instead of plain text:

```vue
<template>
  <Accordion :items="items">
    <template #custom-content>
      <div class="space-y-4">
        <img 
          src="https://placehold.co/600x400" 
          alt="Example image" 
          class="w-full rounded-lg"
        />
        <p class="text-sm text-muted">
          This is custom content rendered through a slot. You can include any Vue components, images, or HTML elements.
        </p>
        <button class="px-4 py-2 bg-primary text-white rounded">
          Learn More
        </button>
      </div>
    </template>
    
    <template #code-example>
      <pre class="bg-neutral-100 dark:bg-neutral-800 p-4 rounded overflow-x-auto"><code>const example = "This is code content";</code></pre>
    </template>
  </Accordion>
</template>

<script setup lang="ts">
import { Accordion } from "@veloce-vue/ui";
import { type AccordionItem } from "@veloce-vue/types";

const items: AccordionItem[] = [
  {
    title: "Image Content",
    slot: "custom-content",
    content: "", // This is ignored when slot is provided
  },
  {
    title: "Code Example",
    slot: "code-example",
    content: "",
  },
];
</script>
```

## Styling

The Accordion component is fully responsive and supports dark mode. It uses CSS variables for colors, making it easy to customize through your theme configuration. The component includes smooth animations for expanding and collapsing content, powered by Motion Vue.

### Customization

You can customize the accordion appearance using:

- **Props**: `rounded`, `shadow`, `bordered` for basic styling
- **CSS Classes**: `contentClass` and `headerClass` for advanced customization
- **Slots**: Use named slots for completely custom content rendering
- **Theme Variables**: Override CSS variables in your theme configuration

### Animation

The accordion uses smooth height and opacity transitions when expanding or collapsing. The animation duration is 300ms with an easing function for a natural feel. The chevron icon rotates 180 degrees when an item is active.
