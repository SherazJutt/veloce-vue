# Accordion

The Accordion component allows you to display collapsible content sections. Users can expand or collapse individual items to reveal or hide content.

## Installation

```javascript
import { Accordion } from "veloce/ui";
```

## Basic Usage

```vue
<template>
  <Accordion :items="items" />
</template>

<script setup lang="ts">
import { Accordion } from "veloce/ui";
import { type AccordionItem } from "veloce/types";

const items: AccordionItem[] = [
  {
    title: "Item 1",
    content: "This is the content for item 1",
  },
  {
    title: "Item 2",
    content: "This is the content for item 2",
  },
  {
    title: "Item 3",
    content: "This is the content for item 3",
  },
];
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `AccordionItem[]` | **required** | Array of accordion items to display |
| `multiple` | `boolean` | `false` | Allow multiple items to be open at the same time |
| `shadow` | `boolean` | `false` | Add shadow to the accordion container |
| `bordered` | `boolean` | `true` | Show borders between items |
| `rounded` | `"none" \| "sm" \| "md" \| "lg" \| "xl"` | `"sm"` | Border radius of the accordion |
| `contentClass` | `string` | `""` | Additional CSS classes for content area |
| `headerClass` | `string` | `""` | Additional CSS classes for header area |

## AccordionItem Type

```typescript
type AccordionItem = {
  title: string;           // Title text displayed in the header
  content: string;         // Content text displayed when expanded
  icon?: Component;        // Optional icon component
  slot?: string;          // Optional slot name for custom content
  active?: boolean;       // Optional initial active state
};
```

### Multiple Open Items

By default, only one accordion item can be open at a time. Set `multiple` to `true` to allow multiple items to be open simultaneously:

```vue
<template>
  <Accordion :items="items" :multiple="true" />
</template>

<script setup lang="ts">
import { Accordion } from "veloce/ui";
import { type AccordionItem } from "veloce/types";

const items: AccordionItem[] = [
  {
    title: "Item 1",
    content: "This can be open at the same time as other items",
    active: true,
  },
  {
    title: "Item 2",
    content: "This can also be open simultaneously",
    active: true,
  },
  {
    title: "Item 3",
    content: "Multiple items can be expanded together",
  },
];
</script>
```

### No Border

Remove borders between items:

```vue
<template>
  <Accordion :items="items" :bordered="false" />
</template>
```

### Custom Rounded Corners

Control the border radius:

```vue
<template>
  <!-- No rounded corners -->
  <Accordion :items="items" rounded="none" />
  
  <!-- Small rounded corners (default) -->
  <Accordion :items="items" rounded="sm" />
  
  <!-- Medium rounded corners -->
  <Accordion :items="items" rounded="md" />
  
  <!-- Large rounded corners -->
  <Accordion :items="items" rounded="lg" />
  
  <!-- Extra large rounded corners -->
  <Accordion :items="items" rounded="xl" />
</template>
```

### With Shadow

Add shadow to the accordion container:

```vue
<template>
  <Accordion :items="items" shadow />
</template>
```

### Custom Content with Slots

Use slots for custom content instead of plain text:

```vue
<template>
  <Accordion :items="items">
    <template #custom-content>
      <div class="p-4">
        <h3>Custom Content</h3>
        <p>This is custom slot content with HTML elements.</p>
        <Button label="Action Button" />
      </div>
    </template>
  </Accordion>
</template>

<script setup lang="ts">
import { Accordion } from "veloce/ui";
import { Button } from "veloce/ui";
import { type AccordionItem } from "veloce/types";

const items: AccordionItem[] = [
  {
    title: "Custom Content",
    slot: "custom-content", // Reference to the slot name
  },
];
</script>
```

## Behavior

- **Single Mode (default)**: Only one item can be open at a time. Opening a new item automatically closes the previously open item.
- **Multiple Mode**: Multiple items can be open simultaneously. Each item can be toggled independently.
- **Animations**: Smooth expand/collapse animations are included using motion-v.
- **Accessibility**: Keyboard navigation and ARIA attributes are supported.

## Styling

The accordion uses Tailwind CSS classes and supports dark mode. You can customize the appearance using the `contentClass` and `headerClass` props or by overriding CSS variables.
