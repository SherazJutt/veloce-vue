## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `"horizontal" \| "vertical"` | `"vertical"` | Direction of the separator |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity |
| `type` | `"solid" \| "dashed" \| "dotted"` | `"solid"` | Border style type |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size/thickness of the separator |
| `label` | `string` | `""` | Label text displayed on the separator |
| `icon` | `Component` | `null` | Icon component to display |
| `bordered` | `boolean` | `false` | Add border around label/icon |
| `rounded` | `boolean` | `false` | Apply rounded corners to label/icon |
| `height` | `string` | `""` | Custom height for vertical separators (e.g., "64px", "1rem") |

## Slots

| Slot | Description |
|------|-------------|
| `content` | Custom content for label/icon area |
