## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `content` | `string` | `""` | Text content to display in the tooltip |
| `position` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | Position of the tooltip relative to trigger |
| `bodyClass` | `string` | `""` | Custom CSS classes for tooltip body |

## Slots

| Slot | Description |
|------|-------------|
| `default` | Trigger element that shows tooltip on hover |
| `body` | Custom tooltip content (overrides `content` prop) |
