## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `"top" \| "bottom"` | `"bottom"` | Position of the popover relative to trigger |
| `bodyClass` | `string` | `""` | Custom CSS classes for popover body |
| `contentClass` | `string` | `""` | Custom CSS classes for popover content |

## Slots

| Slot | Description |
|------|-------------|
| `default` | Trigger element that shows popover on hover |
| `body` | Popover content |
