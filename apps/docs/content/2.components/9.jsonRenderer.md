## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `json` | `JsonRendererItem[]` | `required` | Array of JSON objects defining components to render |

## Supported Components

The JsonRenderer supports the following component types:

- **Typography**: `h1`, `h2`, `h3`, `h4`, `h5`, `h6`, `p`, `span`, `blockquote`, `code`, `label`
- **Lists**: `list` (with `type: "ul" | "ol"`)
- **Messages**: `message` (with severity and variant props)

Each component accepts props based on its type. See the [JsonRendererItem type definition](/types) for detailed prop information.
