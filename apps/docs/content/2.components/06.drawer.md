## Directions

::

::code-collapse
  ```vue
  <template>
    <Drawer v-model="show" direction="left" title="Left Drawer" description="This drawer slides in from the left">
      <Button label="Open Left Drawer" />
      <template #content>
        <p class="text-muted">Content for left drawer</p>
      </template>
    </Drawer>
  </template>
  
  <script setup lang="ts">
  ```

## With Footer

::DrawerContent{:withFooter="true"}
::

::code-collapse
```vue
<template>
  <Drawer v-model="show" direction="bottom" title="Drawer with Footer" description="This drawer has a footer section" :show-footer="true">
    <Button label="Open Drawer" />
    <template #content>
      <p class="text-muted">Drawer content goes here</p>
    </template>
    <template #footer>
      <div class="flex justify-end gap-2">
        <Button label="Cancel" variant="soft" @click="show = false" />
        <Button label="Save" @click="show = false" />
      </div>
    </template>
  </Drawer>
</template>

<script setup lang="ts">
import { Drawer, Button } from "veloce-vue/ui";
import { ref } from "vue";

const show = ref(false);
</script>
```
::

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `boolean` | `false` | Two-way binding for drawer visibility |
| `direction` | `"left" \| "bottom" \| "right"` | `"bottom"` | Direction from which the drawer slides in |
| `title` | `string` | `""` | Title displayed in the drawer header |
| `description` | `string` | `""` | Description text displayed below the title |
| `showFooter` | `boolean` | `false` | Show footer section with footer slot |

## Slots

| Slot | Description |
|------|-------------|
| `default` | Trigger element that opens the drawer when clicked |
| `content` | Main content area of the drawer |
| `footer` | Footer section (only shown when `showFooter` is `true`) |
