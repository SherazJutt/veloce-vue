## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity of the spinner |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of the spinner |
