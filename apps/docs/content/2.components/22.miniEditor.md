## Formatting

The MiniEditor provides a toolbar with three formatting options:

- **Bold**: Make selected text bold
- **Italic**: Make selected text italic
- **Highlight**: Highlight selected text with a background color

::MiniEditorContent{:formatting="true"}
::

::code-collapse
```vue
<template>
  <MiniEditor v-model="content" />
</template>

<script setup lang="ts">
import { MiniEditor } from "veloce-vue/ui";
import { ref } from "vue";

const content = ref("Select this text and format it using the toolbar buttons.");
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `string` | `""` | Two-way binding for editor content (HTML string) |

## Features

- **Rich Text Editing**: Content editable div with formatting capabilities
- **Toolbar**: Built-in toolbar with Bold, Italic, and Highlight buttons
- **State Management**: Toolbar buttons reflect the current formatting state of selected text
- **HTML Output**: Returns HTML string that can be stored and displayed

## Notes

- The component uses `contenteditable` for rich text editing
- Content is stored as HTML string
- Toolbar buttons automatically update based on the current selection
- Highlight formatting can be toggled on/off by selecting highlighted text and clicking the highlight button again
