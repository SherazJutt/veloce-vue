# Badge

The Badge component is used to display small pieces of information like notification counts, status indicators, or labels. It supports both numeric values and dot indicators.

## Installation

```javascript
import { Badge } from "veloce/ui";
```

## Basic Usage

```vue
<template>
  <Badge value="5" />
</template>

<script setup lang="ts">
import { Badge } from "veloce/ui";
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string \| number` | `""` | The value to display in the badge |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity of the badge |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Size of the badge |
| `dot` | `boolean` | `false` | Display as a dot indicator instead of value |

## Examples

### Basic Badges

Display badges with different severity colors:

```vue
<template>
  <div class="flex gap-4">
    <Badge value="2" />
    <Badge value="6" severity="secondary" />
    <Badge value="8" severity="success" />
    <Badge value="4" severity="info" />
    <Badge value="9" severity="warning" />
    <Badge value="3" severity="error" />
    <Badge value="5" severity="neutral" />
  </div>
</template>

<script setup lang="ts">
import { Badge } from "veloce/ui";
</script>
```

### Sizes

Control the size of the badge:

```vue
<template>
  <div class="flex items-center gap-4">
    <Badge value="2" size="sm" />
    <Badge value="6" size="md" />
    <Badge value="8" size="lg" />
    <Badge value="99" size="xl" />
  </div>
</template>
```

### Dot Badges

Display a simple dot indicator without text:

```vue
<template>
  <div class="flex gap-4">
    <Badge dot severity="primary" />
    <Badge dot severity="secondary" />
    <Badge dot severity="success" />
    <Badge dot severity="info" />
    <Badge dot severity="warning" />
    <Badge dot severity="error" />
    <Badge dot severity="neutral" />
  </div>
</template>
```

### Dot Sizes

Control the size of dot badges:

```vue
<template>
  <div class="flex items-center gap-4">
    <Badge dot size="sm" severity="primary" />
    <Badge dot size="md" severity="success" />
    <Badge dot size="lg" severity="info" />
    <Badge dot size="xl" severity="warning" />
  </div>
</template>
```

### With Text

Use badges alongside text labels:

```vue
<template>
  <div class="flex items-center gap-4">
    <span>Notifications</span>
    <Badge value="5" severity="error" />
    
    <span>Messages</span>
    <Badge value="12" severity="info" />
    
    <span>Updates</span>
    <Badge value="3" severity="success" />
  </div>
</template>
```

### Custom Content

Use slots to display custom content:

```vue
<template>
  <div class="flex gap-4">
    <Badge severity="primary">
      <span class="text-xs">New</span>
    </Badge>
    
    <Badge severity="success">
      <span class="px-4 text-xs">Sale</span>
    </Badge>
    
    <Badge severity="warning">
      <span class="h-8 p-2 text-xs">Hot</span>
    </Badge>
  </div>
</template>
```

## Use Cases

### Notification Counts

```vue
<template>
  <button class="relative">
    <Icon :icon="Bell" />
    <Badge 
      value="5" 
      severity="error" 
      class="absolute -top-2 -right-2"
    />
  </button>
</template>

<script setup lang="ts">
import { Badge } from "veloce/ui";
import { Bell } from "veloce/icons";
</script>
```

### Status Indicators

```vue
<template>
  <div class="flex items-center gap-2">
    <Badge dot severity="success" />
    <span>Online</span>
  </div>
</template>
```

### Labels

```vue
<template>
  <div>
    <h2>Featured Products</h2>
    <Badge severity="primary">Featured</Badge>
  </div>
</template>
```

## Styling

Badges are fully responsive and support dark mode. The component uses CSS variables for colors, making it easy to customize through your theme configuration.
