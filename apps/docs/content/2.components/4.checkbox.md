## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `v-model` | `boolean` | `false` | Two-way binding for checkbox state |
| `label` | `string` | `""` | Label text displayed next to the checkbox |
| `labelClass` | `string` | `""` | Custom CSS classes for the label |
| `checkboxClass` | `string` | `""` | Custom CSS classes for the checkbox |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size of the checkbox |
| `variant` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error"` | `"primary"` | Color variant of the checkbox |
| `icon` | `Component` | `Check` | Icon component to display when checked |
| `disabled` | `boolean` | `false` | Disable the checkbox |
