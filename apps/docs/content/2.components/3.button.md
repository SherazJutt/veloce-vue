# Button

The Button component is one of the most versatile components in VeloceVue. It supports multiple variants, severities, sizes, icons, and states to fit any use case.

## Installation

```javascript
import { Button } from "veloce/ui";
```

## Basic Usage

```vue
<template>
  <Button label="Click Me" />
</template>

<script setup lang="ts">
import { Button } from "veloce/ui";
</script>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `""` | Button text label |
| `variant` | `"solid" \| "soft" \| "outlined" \| "ghost" \| "link"` | `"solid"` | Visual style variant |
| `severity` | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error" \| "neutral"` | `"primary"` | Color severity |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | Button size |
| `icon` | `Component` | `null` | Icon component to display |
| `iconPosition` | `"left" \| "right"` | `"right"` | Position of the icon relative to label |
| `rounded` | `boolean` | `false` | Apply full border radius (pill shape) |
| `loading` | `boolean` | `false` | Show loading state with spinner |
| `disabled` | `boolean` | `false` | Disable the button |
| `fontWeight` | `"normal" \| "medium" \| "semibold" \| "bold"` | `"medium"` | Font weight of the label |
| `neumorphic` | `boolean` | `false` | Apply neumorphic styling (requires soft variant) |
| `highlighted` | `boolean` | `false` | Apply highlight background |

## Variants

### Solid

The default variant with filled background:

```vue
<template>
  <Button variant="solid" label="Solid Button" />
</template>
```

### Soft

A softer variant with lighter background:

```vue
<template>
  <Button variant="soft" label="Soft Button" />
</template>
```

### Outlined

Button with border and transparent background:

```vue
<template>
  <Button variant="outlined" label="Outlined Button" />
</template>
```

### Ghost

Minimal button with no background or border:

```vue
<template>
  <Button variant="ghost" label="Ghost Button" />
</template>
```

### Link

Text-only button styled like a link:

```vue
<template>
  <Button variant="link" label="Link Button" />
</template>
```

## Severities

Combine variants with different color severities:

```vue
<template>
  <div class="flex gap-3">
    <Button severity="primary" label="Primary" />
    <Button severity="secondary" label="Secondary" />
    <Button severity="success" label="Success" />
    <Button severity="info" label="Info" />
    <Button severity="warning" label="Warning" />
    <Button severity="error" label="Error" />
    <Button severity="neutral" label="Neutral" />
  </div>
</template>
```

## Rounded Buttons

Apply full border radius for pill-shaped buttons:

```vue
<template>
  <div class="flex gap-3">
    <Button label="Default" />
    <Button rounded label="Rounded Full" />
    <Button :icon="Check" rounded severity="success" />
    <Button :icon="Sun" rounded severity="neutral" />
  </div>
</template>
```

## Font Weights

Control the font weight of the button label:

```vue
<template>
  <div class="flex gap-3">
    <Button font-weight="normal" label="Normal" />
    <Button font-weight="medium" label="Medium" />
    <Button font-weight="semibold" label="Semibold" />
    <Button font-weight="bold" label="Bold" />
  </div>
</template>
```

## Neumorphic Style

Apply neumorphic styling (works best with soft variant):

```vue
<template>
  <div class="flex gap-3">
    <Button neumorphic label="Neumorphic" variant="soft" severity="neutral" />
    <Button neumorphic label="Neumorphic" variant="soft" severity="primary" />
    <Button neumorphic label="Neumorphic" variant="soft" severity="secondary" />
  </div>
</template>
```

## Custom Content (Slots)

Use slots for custom button content:

```vue
<template>
  <div class="flex gap-3">
    <Button>
      <span class="font-bold">Custom</span>
      <span class="text-xs opacity-75">Content</span>
    </Button>
    
    <Button variant="outlined">
      <span>Custom</span>
      <span class="text-xs">Slot</span>
    </Button>
  </div>
</template>
```

## Event Handling

```vue
<template>
  <Button 
    label="Click Me" 
    @click="handleClick"
  />
</template>

<script setup lang="ts">
import { Button } from "veloce/ui";

const handleClick = () => {
  console.log("Button clicked!");
};
</script>
```

## Accessibility

The Button component includes:
- Proper ARIA attributes
- Keyboard navigation support
- Focus states
- Disabled state handling

## Styling

Buttons are fully responsive and support dark mode. All colors use CSS variables from your theme configuration, making customization easy.
