## Getting Started

Docus makes it easy to create beautiful documentation sites. Here's how to get started:

### Write Your Content

Create markdown files in the `content/` directory. Docus will automatically generate routes and navigation from your file structure.

### Customize Your Site

Configure your site using `app.config.ts` to customize colors, branding, and other settings.

### Deploy

Build your site and deploy it to any static hosting provider.
