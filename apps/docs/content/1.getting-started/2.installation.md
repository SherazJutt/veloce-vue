## Import Styles

Import the library's CSS file in your main application file (e.g., `main.ts` or `main.js`):

```javascript
import "veloce/style.css";
```

## Import Components

Import the components you need in your Vue components:

```javascript
import { Button } from "veloce/ui";
```

## Complete Setup Example

Here's a complete example of setting up VeloceVue in a Vue 3 application:


```typescript [main.ts]
import { createApp } from "vue";
import App from "./App.vue";

// Import VeloceVue styles
import "veloce/style.css";

const app = createApp(App);
app.mount("#app");
```

```vue [pages/index.vue]
<template>
  <div>
    <Button label="Click Me" severity="primary" />
  </div>
</template>

<script setup lang="ts">
import { Button } from "veloce/ui";
</script>
```
