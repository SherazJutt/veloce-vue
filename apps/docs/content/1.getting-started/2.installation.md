:::code-group

```bash [pnpm]
pnpm add veloce-vue
```

```bash [yarn]
yarn add veloce-vue
```

```bash [npm]
npm install veloce-vue
```

```bash [bun]
bun add veloce-vue
```

:::


## Import Styles

Import the library's CSS files in your application:

```typescript [style.css]
@import "node_modules/veloce-vue/styles/index.css";
@source "node_modules/veloce-vue/components";
```

## Import Components

Import the components you need in your Vue components:

```typescript
import { Button } from "veloce/ui";
```
