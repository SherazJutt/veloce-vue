# Installation

This guide will walk you through installing and setting up VeloceVue in your Vue 3 application.

## Install the Package

Install VeloceVue using your preferred package manager:

```bash
pnpm add veloce
```

```bash
npm install veloce
```

```bash
yarn add veloce
```

## Import Styles

Import the library's CSS file in your main application file (e.g., `main.ts` or `main.js`):

```javascript
import "veloce/style.css";
```

## Import Components

Import the components you need in your Vue components:

```javascript
import { Button } from "veloce/ui";
```

## Complete Setup Example

Here's a complete example of setting up VeloceVue in a Vue 3 application:

### main.ts

```typescript
import { createApp } from "vue";
import App from "./App.vue";

// Import VeloceVue styles
import "veloce/style.css";

const app = createApp(App);
app.mount("#app");
```

### Component Usage

```vue
<template>
  <div>
    <Button label="Click Me" severity="primary" />
  </div>
</template>

<script setup lang="ts">
import { Button } from "veloce/ui";
</script>
```

## Import Multiple Components

You can import multiple components at once:

```javascript
import { Button, Input, Modal, Badge } from "veloce/ui";
```

## Import Typography Components

Typography components are available from a separate export:

```javascript
import { H1, H2, H3, P, Code, Label } from "veloce/typography";
```

## Import Icons

Icons are available from the icons export:

```javascript
import { Check, Close, Info, Alert } from "veloce/icons";
```

## Import Composables

Composables like `useToast` and `useColorMode` are available:

```javascript
import { useToast, useColorMode } from "veloce/composables";
```
