## Creating the Configuration File

Create a `filename.ts` file in your project with the following structure:

```typescript
import { config } from "veloce/config";

const themeConfig = {
  colors: {
    // Main Brand Colors
    primary: string | "#0060df",
    secondary: string | "#e53e3e",
  },
};

export default config(themeConfig);
```

Import and use the configuration in your main application file:

```typescript [main.ts]
import UIConfig from "./filename";

// Apply UI configuration
app.use(UIConfig);
```

## Using CSS Variables

The configured colors are available as CSS variables:

```css
.custom-element {
  background-color: var(--colors-ui-primary);
  color: var(--ui-color-secondary);
}
```

Or in Vue templates:

```vue
<template>
  <div :style="{ backgroundColor: 'var(--colors-ui-primary)' }">
    Custom styled element
  </div>
</template>
```

## Available options

**colors**: Object containing the colors for the theme.

| Property | Type | Default Value | Description |
|----------|------|---------------|-------------|
| `primary` | `string` | `"#0060df"` | The primary color for the theme. |
| `secondary` | `string` | `"#e53e3e"` | The secondary color for the theme. |
| `success` | `string` | `"#00c271"` | The success color for the theme. |
| `info` | `string` | `"#2b7fff"` | The info color for the theme. |
| `warning` | `string` | `"#f0b100"` | The warning color for the theme. |
| `error` | `string` | `"#fb2c36"` | The error color for the theme. |
| `neutral` | `string` | `"#e5e7eb"` | The neutral color for the theme. |
