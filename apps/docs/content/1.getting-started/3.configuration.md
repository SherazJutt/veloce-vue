## Creating the Configuration File

Create a `ui.config.ts` file in your project root with the following structure:

```typescript
import { config } from "veloce/config";

const themeConfig = {
  colors: {
    // Main Brand Colors
    primary: string | "#0060df",
    "primary-light": string | "#378cff",
    secondary: string | "#e53e3e",
    "secondary-light": string | "#ff7878",

    // default colors
    success: string | "#14b8a6",
    "success-light": string | "#43e6d8",
    info: string | "#3b82f6",
    "info-light": string | "#90cdf4",
    warning: string | "#eab308",
    "warning-light": string | "#fde68a",
    error: string | "#ef4444",
    "error-light": string | "#fca5a5",
    neutral: string | "#64748b",
    "neutral-light": string | "#e5e7eb",
  },
};

export default config(themeConfig);
```

Import and use the configuration in your main application file:

```typescript [main.ts]
import UIConfig from "./ui.config";

// Apply UI configuration
app.use(UIConfig);
```

## Using CSS Variables

The configured colors are available as CSS variables:

```css
.custom-element {
  background-color: var(--ui-color-primary);
  color: var(--ui-color-secondary);
}
```

Or in Vue templates:

```vue
<template>
  <div :style="{ backgroundColor: 'var(--ui-color-primary)' }">
    Custom styled element
  </div>
</template>
```
